A list of git commands

My first repository

  • git clone {url}

    Clone a github repository to local machine

  • git status

    Get the status off all the files

  • git add {file name}, {.} {-A}

    Add file by name, Add full directory, Add all

  • git commit -m "{message}"

    Add a comment (what did you change)

  • git push

    Push the added files to the repository

  • git pull

    Pull the files from the repository
